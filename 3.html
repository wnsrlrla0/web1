JavaScript란 무엇인가?

자바 스크립트 ( / dʒ ɑː V ə ˌ S K R ɪ P t / )
종종 약칭 JS , 인 프로그래밍 언어 것을 따를 대한 ECMAScript 사양.
자바 스크립트는 고수준 , 종종 Just-In-Time 컴파일 및 다중 패러다임 입니다.
이 갖는 곱슬 브래킷 구문 , 동적 타이핑 , 프로토 타입 기반 객체 지향 및 일류 기능.
HTML 및 CSS 와 함께 JavaScript는 World Wide Web 의 핵심 기술 중 하나입니다 .
웹 사이트의 97 % 이상 이 웹 페이지 동작을 위해 클라이언트 측 에서 사용 하며
종종 타사 라이브러리를 통합 합니다.
모든 주요 웹 브라우저는 전용이 자바 스크립트 엔진 상의 코드를 실행하는 사용자 의 장치.
다중 패러다임 언어 인 JavaScript는 이벤트 기반 , 기능적 , 명령형 프로그래밍 스타일을 지원 합니다.
여기에는 텍스트, 날짜, 정규식 , 표준 데이터 구조 및 DOM ( 문서 개체 모델 ) 작업을위한 API ( 응용 프로그래밍 인터페이스 )가 있습니다.
ECMAScript 표준에는 네트워킹 , 스토리지 또는 그래픽 기능 과 같은 입 / 출력 (I / O)이 포함되지 않습니다 .
실제로 웹 브라우저 또는 기타 런타임 시스템 은 I / O 용 JavaScript API를 제공합니다.
